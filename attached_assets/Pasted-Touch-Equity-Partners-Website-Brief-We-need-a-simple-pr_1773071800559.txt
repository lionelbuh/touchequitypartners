Touch Equity Partners – Website Brief

We need a simple professional website for Touch Equity Partners.

The site should be very clean, modern, and minimal, similar to a simple investor or venture firm landing page.

Important structure:

The main website must be one single scrolling page

The Customer Login page is separate

After login, users access a Customer Dashboard

There must also be an Admin Page to post updates that appear on the dashboard.

I will provide the logo, so leave a placeholder for it in the header.

The website must be mobile friendly, fast, and simple to maintain.

Suggested stack:

Next.js or simple React

Supabase for authentication and database

1. Pages Structure

The website will have four pages total.

Main Page (Single Scrolling Page)

Customer Login Page

Customer Dashboard (after login)

Admin Page (restricted access)

5. Admin Page

Create a simple admin page to manage messages displayed on the dashboard.

This page should only be accessible to admin users.

Use Supabase authentication to control access.

Example rule:

users.role = admin

If the user is not admin → deny access.

Admin Features

Admin should be able to:

Create new dashboard posts
Edit posts
Delete posts
Publish or unpublish posts

Admin Form Fields

Title
Message / Content
Optional link
Publish button

Example message:

Title: Welcome to the platform
Message: Here are the first steps to prepare for fundraising.

When published, the message should appear on the customer dashboard.
